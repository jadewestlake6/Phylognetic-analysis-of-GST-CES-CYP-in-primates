# Phylogenetic Analysis of GST, CES, and CYP in Primates

This repository contains sequence data, alignments, and phylogenetic analyses for the GST, CES, and CYP gene families in primates.

### Repository Structure
```
Original_Query_Datasets/
    CES_Original_Query.fasta
    CYP_Original_Query.fasta
    GST_Original_Query.fasta
Primate_CES_Files/
    Primate_CES_GlobalAlignment.fasta
    Primate_CES_LocalAlignment.fasta
    Primate_CES_LocalAlignment.fasta.treefile
Primate_CYP_Files/
    Primate_CYP_LocalAlignment.fasta
    Tree_CYP_Fast.tree
Primate_GST_Files/
    GST_aligned.fasta.treefile
    Primate_GST_LocalAlignment.fasta
FinalArticle.pdf
```
### Folder descriptions

- **Original_Query_Datasets/** – raw FASTA sequences used as input.
- **Primate_CES_Files/** – aligned CES sequences and tree files.
- **Primate_CYP_Files/** – aligned CYP sequences and tree files.
- **Primate_GST_Files/** – aligned GST sequences and tree files.
- **FinalArticle.pdf** – manuscript describing the analyses.
